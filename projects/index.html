Pidovemon: Red


Project Page

The first of the pidovemon romhacks and easily the most scuffed (so-far) made without any understanding of romhacks, git or the codebase. The fact that this exists is enough to make anyone question their faith. Feature wise this romhack adds the gen 5 pokemon pidove to either red or blue, it also adds multiple moves in order to buff pidoves move-set, most are learnt through level-up with a select few becoming available at poke-marts in certain cities. As with all of the early romhacks this was made using pret's decompilations. Where it goes wrong is in the execution, the initial romhack was made simply by downloading source and editing and building entirely locally, while this worked it meant that come time to release this hack to the public there were a myriad of problems, a misunderstanding on my part into the operations of pret had me believing I should fork the repository (this is not what should be done instead it should be added as a seperate upstream.) and then implement my changes, due to me not knowing how git worked at the time, I simply forked the latest version and then went around forcing my old changes into working within the new code base. Do not ask me how it functions, i do not know, I stayed up for 24 hours and then it started building. I haven't touched it since.


Pidovemon: Crystal


Project Page

After the disaster that was publishing red surely I learn't my lesson with crystal right? Kinda... Crystal's release is certainly an improvement in that I didn't fork the repo and force my changes. However I had once again made the entire project locally, (this wasn't me failing to learn my lesson as much as it was me beginning work immediately after finishing red.) Because of this I just took my local files and dumped them into a github repo and called it a day. This one functions and is significantly better than Red, but it still isn't done properly. Feature wise once again it adds pidove and some learnset moves. This one does not add any new TM's. Once again would not be possible without the brilliant work of pret's decompilations.


Pidovemon: Emerald


Project Page

You wont believe it, but this one was actually made correctly, the repo is properly connected to pret's pokeemerald decomp. YIPPEE. Feature wise this is the most feature rich, not only is pidove added but so are their abilities, as well as allowing all HM's to be used from the bag in order to make solo runs easier. Unlike previous entries no new moves are added.


Mad Verse City: Cleaned Up Prompts

Project Page

A mod I quickly put together for Mad Verse City from The Jackbox Party Pack 5, after multiple frineds complained about how the prompts were far too vague as to ruin the experience for them, put together in less than 24 hours this project comes down to editing 3 json files. Still it does everything it needs to do. After just over 2 years of being a project only available in a small community discord server, now it is available through github.
